swf_posround -- Enables or Disables the rounding of the translation when
objects are placed or moved

  void swf_posround(int round)~

Enables or disables the rounding of the translation when objects are placed or
moved, there are times when text becomes more readable because rounding has
been enabled.

{round} Whether to enable rounding or not, if set to the value of 1, then
rounding is enabled, if set to 0 then rounding is disabled.

No value is returned.

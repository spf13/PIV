tidy_load_config -- Load an ASCII Tidy configuration file with the specified
encoding

  void tidy_load_config(string filename, string encoding)~

Loads a Tidy configuration file, with the specified {encoding}.

{filename}

{encoding}

No value is returned.

This function is only available in Tidy 1.0. It became obsolete in Tidy 2.0,
and thus has been removed.

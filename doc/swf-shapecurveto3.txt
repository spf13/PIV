swf_shapecurveto3 -- Draw a cubic bezier curve

  void swf_shapecurveto3(float x1, float y1, float x2, float y2, float x3, float y3)~

Draw a cubic bezier curve using the given coordinates.

The current position is then set to the {x3},{y3} coordinate.

{x1} x-coordinate of the first off curve control point.

{y1} y-coordinate of the first off curve control point.

{x2} x-coordinate of the second off curve control point.

{y2} y-coordinate of the second off curve control point.

{x3} x-coordinate of the endpoint.

{y3} y-coordinate of the endpoint.

No value is returned.

|swf_shapecurveto|

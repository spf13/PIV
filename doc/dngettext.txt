dngettext -- Plural version of dgettext

  string dngettext(string domain, string msgid1, string msgid2, int n)~

The |dngettext| function allows you to override the current {domain} for a
single plural message lookup.

{domain} The domain

{msgid1}

{msgid2}

{n}

A string on success.
